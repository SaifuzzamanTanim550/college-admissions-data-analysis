# College Admissions Data Analysis

This project explores a freshman admissions dataset using **Python and Tableau**.  
The goal is to clean raw admissions data, analyze important patterns, and present the results through an **interactive Tableau dashboard**.

The dashboard helps explain trends in applications, admission outcomes, academic indicators, and geographic distribution of accepted students.

**Interactive Tableau Dashboard**  
https://public.tableau.com/app/profile/saifuzzaman.tanim/viz/BrooklynCollegeAdmissionAnalysisDev4/FinalDashboard

# Dataset Overview

The dataset contains **9,007 freshman applicants**.

From those applicants:

- **5,920 students were admitted**
- **3,087 students were denied**
- **Acceptance rate: 65.7%**

The dataset includes information such as:

- GPA  
- Gender  
- Ethnicity  
- High school attended  
- ZIP code  
- Admission decision  

---

# Analysis Process

### Data Cleaning

The raw dataset required cleaning and preparation before analysis.  
Python and Pandas were used to organize the data, correct formatting issues, and prepare fields for analysis.

### Exploratory Data Analysis

Exploration of the dataset focused on patterns such as:

- admission and denial outcomes  
- GPA ranges and acceptance rates  
- demographic distribution of applicants  
- high schools sending the most applicants  
- geographic distribution of accepted students  

### Tableau Dashboard

The results are presented in an **interactive Tableau dashboard** that allows users to explore the data through multiple visual sections.

---

# Dashboard Sections

### Application Volume and Acceptance Outcomes

Shows the total number of applicants, admitted students, denied students, and the overall acceptance rate.

### Gender and Ethnicity Distribution

Compares demographic distributions between applicants and admitted students.

### GPA and Admission Outcomes

Shows how acceptance rates increase as GPA ranges rise.

### High Schools with the Most Applications

Highlights high schools sending the largest number of applicants.

### High Schools with the Highest Acceptance Rates

Shows schools where applicants have strong admission outcomes.

### Geographic Distribution

A map showing where accepted students come from based on ZIP codes.
